Ce projet de machine learning a été réalisé par Ibrahim Lmourid et Clément Préaut.

On trouve dans ce dépôt une feuille Jupyter (goProject.ipynb) contenant toutes les fonctions qui mettent en place un modèle de Machine Learning, ainsi que l'homologue de ce fichier en HTML.
En exécutant toutes les feuilles dans l'ordre, on structure les données, on crée et entraîne le modèle. Enfin dans la dernière feuille, on récupère le fichier d'évalation et on effectue les prédictions pour celui-ci.

Les prédictions sont stockées dans le fichier my_predictions.txt.
